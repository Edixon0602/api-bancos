API códigos bancarios

Esta API ofrece dos endpoints para la obtención de informacion de los bancos nacionales con el uso de sus códigos

Endpoints

GET /all

Retorna todos los bancos

https://api-bancos.edixonserrano.com/all

GET /bank/:code

Retorna la informacion de un banco especifico utilizando el código como parámetro

https://api-bancos.edixonserrano.com/bank/{codigo}
